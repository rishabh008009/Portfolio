# 🚀 Rishabh Singh — Product Manager & Agentic AI Portfolio

A premium animated portfolio showcasing 8+ years of Product Management and Operations excellence at Amazon, combined with hands-on Agentic AI product development.

🔗 **Live Site**: [rishabh-singh-pm.vercel.app](https://rishabh-singh-pm.vercel.app)

---

## ✨ Features

- **Interactive Particle Background** — Mouse-reactive constellation particles
- **Animated Stats Counter** — Key metrics that count up on scroll
- **3D Tilt Project Cards** — Hover-responsive cards with shimmer borders
- **Smooth Scroll Animations** — Staggered entrance animations powered by Framer Motion
- **Dark Glassmorphism Design** — Premium purple-accented dark theme
- **Fully Responsive** — Optimized for desktop, tablet, and mobile

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **Next.js 16** | React framework with SSR |
| **React 19** | UI components |
| **Framer Motion** | Animations & transitions |
| **CSS Modules** | Scoped styling |
| **Lucide React** | Icon library |
| **Vercel** | Deployment & hosting |

